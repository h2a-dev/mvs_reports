# MVS Reports

DeFi supply opportunity reports generated by the Most Valuable Strategy discovery tool.

## Structure

```
├── aave/
│   └── aave_opportunities_report.md
├── fluid/
│   └── fluid_opportunities_report.md
├── morpho/
│   └── morpho_opportunities_report.md
└── unified_mvs_report.md
├── defi_arsenal_stablecoins.md
└── defi_arsenal_collateral.md
```

## Reports

| Report | Description |
|--------|-------------|
| `aave_opportunities_report.md` | Aave lending opportunities with Merkle rewards |
| `fluid_opportunities_report.md` | Fluid protocol supply rates |
| `morpho_opportunities_report.md` | Morpho vault opportunities (v1 + v2) |
| `unified_mvs_report.md` | Combined report ranked by Total APY |
| `defi_arsenal_stablecoins` | List of low-risk stablecoins to Supply used to filter out high-risk opportunities |
| `defi_arsenal_collateral.md` | List of low-risk crypto assets that borrowers can use as a collateral to filter out high-risk opportunities |

Reports are published to [h2a-dev/mvs_reports](https://github.com/h2a-dev/mvs_reports).
