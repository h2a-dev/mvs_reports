# MVS Reports

DeFi supply opportunity reports generated by the Multi-Protocol Vault Scanner (MVS) pipeline.

## Structure

Each pipeline run creates a timestamped directory:

```
reports/
├── 2026-01-30T14-30-00/
│   ├── aave/
│   │   └── aave_opportunities_report.md
│   ├── fluid/
│   │   └── fluid_opportunities_report.md
│   ├── morpho/
│   │   └── morpho_opportunities_report.md
│   └── unified_mvs_report.md
└── ...
```

## Reports

| Report | Description |
|--------|-------------|
| `aave_opportunities_report.md` | Aave lending opportunities with Merkle rewards |
| `fluid_opportunities_report.md` | Fluid protocol supply rates |
| `morpho_opportunities_report.md` | Morpho vault opportunities (v1 + v2) |
| `unified_mvs_report.md` | Combined report ranked by Total APY |

## Usage

Generate reports:
```bash
cd app/server && uv run python main.py --verbose
```

Reports are also published to [h2a-dev/mvs_reports](https://github.com/h2a-dev/mvs_reports).
